# Benchmark Results

Median requests/sec across repeats, per stack × endpoint × load stage, for each machine the benchmark has been run on. Generated by `bench/gen_results.py` from the raw wrk reports under `results/<host>/raw/`.

¹ **Worker stacks** (Doppar custom, Laravel Octane, Symfony native) all run on the **same FrankenPHP base image with 16 workers**, so they ARE comparable *among themselves*. They are NOT like-for-like with the nginx+PHP-FPM rows — a worker keeps the app in memory and uses FrankenPHP/Caddy instead of nginx+php-fpm, changing both the web server and the runtime model. What each framework officially provides differs: Doppar ships no worker runtime (custom boot-once script), Laravel uses first-party Octane (`--server=frankenphp`), Symfony uses its native FrankenPHP worker support (symfony/runtime ≥ 7.4). See FAIRNESS.md.
